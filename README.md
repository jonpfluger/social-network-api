# Social Network API

## Description

API for a social network web application where users can create a friends list, share their thoughts, and react to friends' thoughts.

## Technologies Used

* MongoDB database
* Mongoose ODM
* Express.js
* JavaScript
    * Native JavaScript Date object to format timestamps
